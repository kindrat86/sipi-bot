Merchant Allowlist

How merchant allowlists protect against unauthorized agent purchases.

A merchant allowlist is a predefined list of approved vendors that an AI agent is permitted to pay. Any transaction to a merchant NOT on the list is automatically blocked. This prevents scenarios where an agent is prompted to 'buy the cheapest option' and purchases from an unknown vendor. Combined with category rules and per-transaction caps, merchant allowlists form the core defense against unauthorized agent spend.
