A typical deployment

  1. List the transactions your agents actually initiate — paid API calls, provisioning, purchases, refunds.
  2. Set a per-transaction limit that covers legitimate use but flags anything unusual. For most LangChain workflows, $1 to $5 per transaction is enough.
  3. Set a daily ceiling per agent ($10-$50 for development, higher for production).
  4. Add a velocity cap (transactions per minute) to catch loops. Ten per minute is a reasonable starting point.
  5. Configure a merchant allowlist so the agent can only transact with vendors you have approved.
  6. Turn on alerts and run a postmortem on every flagged transaction.
